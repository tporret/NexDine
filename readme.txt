== Description ==

This plugin provides the core NexDine bootstrap and extensible architecture.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nexdine` directory, or install via Git deployment.
2. Activate the plugin through the WordPress admin plugins screen.
